# DocDriv

DocDriv is your personal file storage and sharing platform, designed for easy access and direct downloads of your files via GitHub Pages.

## Features

- **Automatic File Listing**: Dynamically lists all files in the repository with details like name, extension, size, and download links.
- **Professional Design**: High-tech dark theme for a modern, sleek interface.
- **Direct Downloads**: Users can download files directly from the web interface.
- **Responsive Layout**: Optimized for desktop and mobile devices.

## How to Use

1. Upload your files to this repository.
2. The landing page (`index.html`) will automatically display all files for download.
3. Share the GitHub Pages URL with your users.

## Setup

To enable GitHub Pages:
1. Go to your repository settings.
2. Under "Pages", select the `main` branch as the source.
3. Your site will be available at `https://zaravtu-cell.github.io/redo/`.

## Supported File Types

DocDriv supports all file types including zip, rar, exe, js, vbs, and more.
